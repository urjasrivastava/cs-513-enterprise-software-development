CS511 Algorithms   by   Prof. Muralidhara
Course Description
This course will cover some of the advanced data structures like Fibonacci Heaps, Treaps, AVL and red black trees, It cover the algorithms design techniques like divide and conquer, greedy algorithms and dynamic programming. It also covers graph algorithms, including shortest path problems and minimum spanning trees and Network flows.
